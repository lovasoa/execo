Overview
========

The `execo_engine` class hierarchy and ``execo-run`` tools provide the
basis for the development of experiment engines. The
`execo_engine.Engine` class is the base class for classes which act as
experiment engines. ``execo_run`` is the launcher for those experiment
engines.
